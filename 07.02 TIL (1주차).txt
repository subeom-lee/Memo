#CSS

background : 배경색
color : 글씨색

background-image -> URL
background-position -> center
background-size -> cover
border : 테두리 -> 1px solid white
border-radius : 귀퉁이 부드럽기 -> px
margin : 바깥여백 -> auto / px

#JAVASCRIPT

변수 : 값을 담는것
자료형 : 숫자인지 문자인지
조건문 : if ~~ else 그게아니면 저렇게된다
반복문 : 계속 반복하자

let a_list -> 리스트 만들기
a_list['num'] -> 리스트 보기 (리스트는 0번부터시작)
a_list.push('a') -> 리스트 추가하기
a_list -> 리스트보기
let a_dict={'name':'bob','age':27} -> 값 설정
a_dict['height'] -> 설정값추가

console.log : 콘솔창에 띄우기
